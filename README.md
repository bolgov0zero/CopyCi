# CopyCi

Minimalist snippet manager for macOS. Lives in your menu bar — paste text snippets anywhere in one click.

## Features

- **Menu bar app** — always accessible, never in the way
- **Snippet picker** appears near your mouse cursor
- **Global hotkey** to show/hide the picker (default: ⌘⇧Space)
- **Sections** to organize snippets
- **Settings** — auto-launch, custom hotkey, add/edit snippets

## Install

Download the latest `.dmg` from [Releases](../../releases), open it and drag CopyCi to Applications.

Grant **Accessibility** permissions in System Settings → Privacy & Security → Accessibility.

## Usage

| Action | Result |
|---|---|
| Left click menu bar icon | Open snippet picker |
| Right click menu bar icon | Settings / Quit |
| ⌘⇧Space (or custom) | Toggle snippet picker |
| Click a snippet | Paste it at cursor position |

## Requirements

macOS 13+
